## 用户登录

用于用户登录的接口



url：

> /login



请求参数表：

|     字段名     | 字段类型 | 是否必须 |          描述          |
| :------------: | :------: | :------: | :--------------------: |
|      uid       |  string  |    否    |         用户ID         |
|     email      |  string  |    否    |        用户邮箱        |
|    password    |  string  |    是    |        用户密码        |
| recaptchaToken |  string  |    是    | recaptcha回传验证token |

备注：

**uid和email必须给其中一项，不需要两项都给，但不能两项都不给**



返回参数表：

|   字段名    | 字段类型 | 是否必须 |        描述         |
| :---------: | :------: | :------: | :-----------------: |
|    code     |   int    |    是    |    无错误则返回0    |
|   message   |  string  |    否    |      错误信息       |
| accessToken |  string  |    否    | 后续用户访问的token |
